YouTube Summarizer

YouTube Summarizer is a web app that uses GPT3.5 to summarize a YouTube video using the video's transcript. Visitors need to provide their own OpenAI API key to use the app. Authentication is done through The Metal Wizard, which allows developers to easily include authentication in their own apps.

Developers can build their own apps with Streamlit, a powerful tool for creating data apps and visualizations. To learn more about Streamlit, visit the official website:
